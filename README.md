React and Django App

Must have:
- Node.js
- Python



Backend steps:
- cd backend
- python -m venv venv
- .\venv\Scripts\Activate.ps1
- pip install django djangorestframework django-cors-headers
- python manage.py migrate
- python manage.py runserver

Frontend steps:
- cd frontend/react
- npm install
- npm run dev

Got to this link to see app:
http://localhost:5173
